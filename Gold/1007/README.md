# boj_1007_vector_matching

### 📌 문제 정보
- **난이도**: Gold 2
- **유형**: 기하학, 브루트포스(조합)
- **문제 링크**: [백준 1007번](https://www.acmicpc.net/problem/1007)

### 💡 풀이 아이디어 (수학적 원리)
- 모든 점을 $N/2$개의 시작점과 $N/2$개의 끝점으로 나누어 벡터를 생성합니다.
- 벡터의 합은 결국 `(끝점들의 합) - (시작점들의 합)`과 같다는 성질을 이용했습니다.
- 전체 좌표의 합을 미리 구한 뒤, 조합(`itertools.combinations`)을 통해 뺄 점들을 골라 계산 효율성을 높였습니다.

### 🛠️ 개발 환경 및 트러블슈팅
- **Language**: Python 3.14
- **IDE**: VS Code (Interactive Window & Terminal 활용)
- **Trouble Shooting**: 
    - 처음에 `input()` 함수 사용 시 VS Code 상단 입력창 버그로 고생했으나, 터미널 실행 방식을 통해 해결했습니다.
    - 점 2개 사이의 거리가 아닌 '전체 벡터 합의 길이'를 구해야 한다는 점을 수식으로 정리하며 로직을 수정했습니다.
